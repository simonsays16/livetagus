Política de Privacidade

Última atualização: 22-01-2026

1. Princípio "Privacy by Design"

A tua privacidade é o pilar central do LiveTagus. A aplicação foi desenhada para funcionar sem saber quem és. Não recolhemos, não vendemos e não processamos dados pessoais identificáveis (PII) nos nossos servidores. Não sabemos o teu nome, a tua localização GPS ou os teus hábitos de viagem.

2. Recolha de Dados e Infraestrutura

O LiveTagus utiliza serviços de terceiros para alojamento e funcionamento técnico, mas limita ao máximo a exposição de dados.

2.1. Alojamento e Backend

  • Frontend: O site é alojado na Netlify.
  • API/Backend: O servidor que processa os dados está alojado em fornecedores de infraestrutura cloud (Azure).
  • Logs Técnicos: Estes fornecedores podem recolher logs técnicos padrão (como endereço IP, User-Agent e data de acesso) estritamente para fins de segurança (prevenção de DDoS), manutenção da rede e diagnóstico de erros. O criador do LiveTagus não utiliza estes logs para rastreamento individual nem os cruza com outras informações.

2.2. Email e Suporte

Se entrares em contacto connosco via email (ex: para reportar bugs):

  • O teu endereço de email e o conteúdo da mensagem serão armazenados apenas pelo tempo estritamente necessário para responder à tua questão e resolver o problema.
  • Após a resolução, os dados são arquivados ou eliminados.
  • Nunca utilizaremos o teu email para listas de marketing, newsletters ou partilha com terceiros.

3. Cookies e Armazenamento Local (LocalStorage)

O LiveTagus não utiliza cookies de rastreio, publicidade comportamental ou analítica invasiva (como Google Analytics). Utilizamos exclusivamente a tecnologia Web LocalStorage do teu navegador. Isto funciona como uma "memória interna" do teu telemóvel, onde guardamos pequenas configurações para que não tenhas de as repetir sempre que abres a app.

Lista detalhada dos dados guardados no teu dispositivo:

Chave (Key) Finalidade Tipo
theme / system Guarda a tua preferência visual (Modo Claro, Escuro ou Sistema). Preferência
pref_margem_org / pref_margem_dest Memoriza as estações de origem/destino escolhidas no sentido Margem Sul. Funcionalidade
pref_lisboa_org / pref_lisboa_dest Memoriza as estações de origem/destino escolhidas no sentido Lisboa. Funcionalidade
pref_sync_stations Define se queres sincronizar as estações entre abas (inverter origem/destino automaticamente). Funcionalidade
ft_tab Memoriza qual a última aba que tinhas aberta (Lisboa ou Margem Sul). Navegação
ft_org / ft_dst Cache temporária para funcionamento da interface. Técnico
enable_smart Guarda a tua preferência sobre a funcionalidade "Smart Schedule" (Horário Inteligente). Configuração
enable_regular Guarda a tua preferência sobre a visualização de horários regulares vs. tempo real. Configuração

Nota Legal: Estes dados são considerados "Estritamente Necessários" para o fornecimento do serviço solicitado pelo utilizador (nos termos da Diretiva ePrivacy e RGPD), dispensando consentimento prévio via banner de cookies. Estes dados residem apenas no teu dispositivo e nunca são enviados para uma base de dados nossa.

  • Podemos ter acesso ao nome associado à doação e ao email (dependendo da plataforma) apenas para fins de confirmação de transação e agradecimento. Estes dados não são inseridos em nenhuma base de dados comercial.
